**Source:** `agentic-learning-plan-v3.md` → Phase 6 (part A)
**Prerequisites:** Sections 1–13 complete
**Estimated time:** 2.5–4 hours (real tier-2 run)

## What you'll walk away knowing

- Tier 2 = multiple Claude instances in isolated worktrees, coordinated by a supervisor agent. Tools: Gas Town, oh-my-claudecode.
- `plan` mode — catches scope issues before any code is written
- The preparation checklist — what must be in order before launching a tier-2 run
- Debrief as mandatory — every run produces at least one new CLAUDE.md rule

## Concepts the teacher will work through with you

1. **Tier 2 is orchestration, not magic.** Solid CLAUDE.md + project skills + clean git = tier-2 viable. Without that = junk.
2. **`plan` mode as cheap insurance.** An hour of interview saves 4 hours of wasted execution.
3. **Worth-it calculus.** A single task is rarely tier-2 worthy. A sprint's worth of well-scoped work is.
4. **Debrief creates the moat.** Every run teaches you something that goes back into CLAUDE.md or your skill library.

- Why is the debrief step non-optional?
